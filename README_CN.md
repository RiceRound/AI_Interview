# 面试辅助系统

[English](README.md)

## 概述
面试辅助系统是一个基于AI的工具，可以将面试官的音频实时转换为文字，并提供合适的回答。系统集成了包括百度、腾讯和Kimi在内的多个AI服务，以获得最佳性能。
